O Scola nasceu como projeto de portfólio para o AI Challenge da FIAP, mas resolve um problema real: a gestão escolar brasileira ainda é fragmentada, manual e reativa.

Com mais de 47 mil escolas públicas no Brasil sem acesso a gestão digital integrada, o mercado é enorme. Nossa abordagem SaaS B2B garante escalabilidade por número de alunos.

Stack: React + FastAPI + PostgreSQL + scikit-learn + Anthropic Claude API. Deploy via Docker Compose em AWS EC2.
